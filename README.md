# 3D-DRAW
Draw in 3D!!!!!!!!!

dependencies: `pip install opencv-python mediapipe`

## VirtualPainter.py
A 2d openCV prototype, allows changing to random color, erasing, and changing between 2 brush size modes

## index.html
Full 3D drawing web file that allows for 3D freehand drawing, powered by three.js. It allows for changing colors, brush size, exporting as gLTF model, straight line snapping, model roation, locking z-axis for more precise drawings, and material options.
